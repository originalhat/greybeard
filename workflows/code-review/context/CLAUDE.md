# Context

## Contents

| File | Purpose |
|------|---------|
| `GOTCHYAS.md` | Known pitfalls and traps in the codebase |
| `NITS.md` | Style preferences and minor conventions |

## Adding Context

Add files here for:
- Repo-specific patterns that reviewers should know
- Team conventions not captured elsewhere
- Historical decisions that affect how code should be written
